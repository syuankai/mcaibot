# 超級無影 (SuperWY)

> 詳細文件請見 [README.md](./README.md)

Minecraft 自動化倉庫管理系統，基於 Node.js + Mineflayer。

## 專案概述

主要功能：
- 倉庫掃描（多區域、斷點續掃、NBT 解析）
- 聊天指令取貨/空投（支援 LLM 自然語言語意解析）
- Litematica 藍圖材料計算
- Web 管理面板（Vue 3 + Element Plus）
- 保活模組（自動補血、擬人行為）

支援的 LLM 提供者：OpenAI、Claude、Gemini、通義千問、百度文心、Azure OpenAI、Ollama。
